
# Favourite Toy

## Overview
A project to manage and showcase your favorite toys with a clean, organized system.

## Features
- Add and organize toy collections
- Track toy details and metadata
- Simple and intuitive interface
- Lightweight and fast

## Installation
```bash
git clone <repository-url>
cd Favourite_Toy
npm install
```
